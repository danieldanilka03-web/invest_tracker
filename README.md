# Invest Tracker

Локальное мобильное приложение для учёта инвестиций. Всё хранится на устройстве (Hive — файловая база, без сервера и без интернета).

## Возможности
- **Покупки** — одна сделка может содержать сразу несколько бумаг (тикер, тип, количество, цена, комиссия, сектор). Автокомплит по встроенному справочнику бумаг Мосбиржи. Удаление свайпом с подтверждением
- **Дивиденды и купоны** — сумма до/после налога, тип, автокомплит по тому же справочнику
- **Планы** — будущие покупки с целевой ценой/количеством/сроком, статус (активен/выполнен/отменён). Удалять план нельзя специально — только менять статус, чтобы случайно не потерять цель
- **Портфель (главный экран)**:
  - крупная карточка текущей стоимости портфеля + график её изменения во времени
  - стоимость каждой бумаги пересчитывается по её последней цене покупки (без интернета/котировок)
  - состав портфеля с переходом на детальный график по каждой бумаге (история цены + список сделок)
  - пирог по секторам, столбцы дохода по месяцам, фильтр периода
- **Настройки** — выбор цветовой палитры приложения (8 готовых вариантов) и темы (светлая/тёмная/авто), история курсов валют по датам, включение/выключение расчёта налога с продаж
- **Виджет на главный экран Android** — стоимость портфеля и прибыль/убыток прямо на рабочем столе телефона, без захода в приложение. Чтобы добавить: зажми пустое место на главном экране → **Виджеты** → найди "Invest Tracker" → перетащи виджет "Портфель" на экран. Данные обновляются при каждом открытии приложения и изменении сделок — сам виджет офлайн и ничего не считает самостоятельно
- **Экспорт JSON** — бэкап всех данных в один файл (важно, раз нет БД-сервера — единственный способ перенести данные на новый телефон)
- **Аватарки бумаг** — цветные иконки с инициалами тикера вместо логотипов (логотипы компаний — чужие товарные знаки, поэтому не встраиваются)

## Как получить готовый APK, ничего не собирая руками

### Вариант A — GitHub Actions (рекомендую, полностью автоматически)
1. Создай пустой репозиторий на GitHub
2. Залей туда содержимое этой папки:
   ```bash
   cd invest_tracker
   git init
   git add .
   git commit -m "init"
   git remote add origin <твой-репозиторий>
   git push -u origin main
   ```
3. Зайди во вкладку **Actions** в репозитории — workflow `Build APK` запустится автоматически
4. Через 3-5 минут в разделе **Artifacts** появится `invest-tracker-apk` — скачай, это готовый файл для установки на Android

Дальше при каждом пуше APK будет пересобираться сам — тебе не нужен ни Android Studio, ни Flutter SDK локально.

### Вариант B — локально (если хочешь iOS-версию или сборку на своей машине)
```bash
flutter pub get
dart run build_runner build --delete-conflicting-outputs
flutter build apk --release      # Android
flutter build ios --release      # iOS (нужен Mac + Xcode)
```

### Вариант C — codemagic.io / appcircle.io
Если хочешь собирать и iOS без Mac — подключи репозиторий к [Codemagic](https://codemagic.io), у них щедрый бесплатный план и есть готовый Flutter-шаблон конфигурации, тоже без единой ручной команды с твоей стороны.

## Установка APK на телефон
Просто перенеси `app-release.apk` на телефон и открой (может потребоваться разрешить "установку из неизвестных источников" в настройках Android).

## Структура проекта
```
lib/
  models/       # Purchase, Income, Plan, AppSettings (Hive-модели); Deposit оставлен для совместимости старых бэкапов
  data/         # Securities — встроенный справочник бумаг Мосбиржи для автокомплита
  services/     # StorageService, AnalyticsService (стоимость портфеля, история цены), ThemeService (палитра), BackupService
  widgets/      # TickerAvatar (аватарка-инициалы), SecurityPickerField (автокомплит бумаги)
  screens/      # HomeScreen, Dashboard, Purchases, Incomes, Plans, Settings, TickerDetail
```

## Важно про хранение данных
Приложение НЕ использует облако и НЕ отправляет данные никуда. Всё лежит локально в песочнице приложения на телефоне. Если удалишь приложение — данные пропадут, если не сделал бэкап через кнопку экспорта на экране статистики (иконка "поделиться" вверху).
